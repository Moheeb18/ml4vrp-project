### Methods used 

- Nearest Neighbor (baseline)
- Clarke-Wright Savings Algorithm

## Optimization Techniques

- Relocate (inter-route improvement)
- Swap (inter-route exchange)
- 2-opt (intra-route optimization)

## Pipeline

Savings → Relocate → Swap → 2-opt

## Datasets

The solution is tested on benchmark VRP datasets (Augerat instances).

## Results

The approach achieves near-optimal solutions with small gaps from the optimal values across all datasets.
